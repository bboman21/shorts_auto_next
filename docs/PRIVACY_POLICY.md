# 개인정보처리방침 (Privacy Policy)

**최종 수정일**: 2026년 1월 11일

---

## 1. 수집하는 정보

**Shorts Auto Next**는 사용자의 개인정보를 수집하지 않습니다.

이 확장 프로그램은 다음과 같은 정보만 로컬에 저장합니다:
- 자동재생 ON/OFF 설정 상태

이 정보는 사용자의 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.

---

## 2. 정보 사용 방법

저장된 설정 정보는 오직 다음 목적으로만 사용됩니다:
- 사용자가 설정한 자동재생 상태를 유지하기 위함

---

## 3. 정보 공유

**Shorts Auto Next**는 어떠한 사용자 정보도 제3자와 공유하지 않습니다.

---

## 4. 데이터 저장

모든 데이터는 Chrome Storage API를 통해 사용자의 로컬 브라우저에만 저장됩니다:
- 데이터는 사용자의 기기를 떠나지 않습니다
- 확장 프로그램 삭제 시 모든 데이터가 함께 삭제됩니다

---

## 5. 권한 사용

이 확장 프로그램은 다음 권한을 사용합니다:

| 권한 | 용도 |
|------|------|
| `storage` | 사용자 설정 저장 |
| `youtube.com` 접근 | YouTube Shorts 페이지에서 자동재생 기능 작동 |

---

## 6. 아동 개인정보 보호

이 확장 프로그램은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

---

## 7. 변경 사항

개인정보처리방침이 변경될 경우, 이 페이지에 업데이트됩니다.

# Privacy Policy (English)

**Last Updated**: January 11, 2026

## Information We Collect

**Shorts Auto Next** does not collect any personal information.

This extension only stores locally:
- Auto-play ON/OFF setting state

This information is stored only in your browser and is never transmitted to external servers.

## How We Use Information

Stored settings are used solely to:
- Maintain your preferred auto-play state

## Information Sharing

**Shorts Auto Next** does not share any user information with third parties.

## Data Storage

All data is stored locally in your browser via Chrome Storage API:
- Data never leaves your device
- All data is deleted when you uninstall the extension

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `storage` | Save user settings |
| `youtube.com` access | Enable auto-play feature on YouTube Shorts |

## Children's Privacy

This extension does not intentionally collect personal information from children under 13.

## Changes

Any changes to this privacy policy will be updated on this page.
