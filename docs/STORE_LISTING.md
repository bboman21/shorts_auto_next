```
🎬 Shorts Auto Next - YouTube Shorts 자동 다음 재생

유튜브 쇼츠를 더 편하게 즐기세요!

📌 왜 Shorts Auto Next인가요?
유튜브 쇼츠는 영상이 끝나면 자동으로 반복 재생됩니다. 
다음 영상을 보려면 매번 직접 넘겨야 하죠.
Shorts Auto Next가 이 불편함을 해결해 드립니다!

✨ 주요 기능
• 자동 다음 재생: 영상이 끝나면 자동으로 다음 쇼츠로 이동
• 간편한 ON/OFF: 화면의 플로팅 버튼으로 쉽게 켜고 끄기
• 설정 저장: 브라우저를 닫아도 설정이 유지됨

👥 이런 분께 추천해요
• 재택근무 중 BGM처럼 쇼츠를 틀어놓는 분
• 식사하면서 손 안 대고 쇼츠 보고 싶은 분  
• 게임 로딩 중 쇼츠 자동 재생 원하는 분
• PC에서 쇼츠를 자주 시청하는 분

🔒 개인정보 보호
• 어떤 개인정보도 수집하지 않습니다
• 모든 설정은 로컬에만 저장됩니다
• 외부 서버 연결 없음

📱 사용 방법
1. 확장 프로그램 설치
2. YouTube Shorts 페이지 방문 (youtube.com/shorts)
3. 우측 하단의 플로팅 버튼으로 ON/OFF 전환
4. 끊김 없이 쇼츠 즐기기!

💡 팁
• 키보드 단축키: 추후 업데이트 예정
• 문제가 있으시면 리뷰나 GitHub에 알려주세요

---
Made with ❤️ for YouTube Shorts lovers
```
